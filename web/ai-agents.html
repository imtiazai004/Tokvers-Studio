AI Agents

Six coordinated agents for a complete content workflow.

The app already exposes agent status in the dashboard. This page explains the role of each agent in plain language for visitors and buyers.

Open Dashboard View Features
1

Research Agent

Looks for market context, trends, product angles, and buyer motivations before the script is written.

2

Script Agent

Turns the product angle into UGC-style structure: hook, pitch, proof, objection handling, and CTA.

3

Voice Agent

Creates voiceover audio through the configured voice provider and selected voice settings.

4

Video Agent

Generates or assembles the visual output using the chosen video engine and campaign inputs.

5

Upload Agent

Supports the publishing workflow and tracks queued upload activity where configured.

6

Optimization Agent

Uses performance history and learnings to improve future creative decisions.

Why Agents

Separate responsibilities make the workflow easier to inspect and improve.

Instead of one opaque generator, the system shows distinct steps. That makes it clearer where a campaign is working, where it is blocked, and what needs better input.
